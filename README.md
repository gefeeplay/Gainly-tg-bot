# Gainly Telegram Bot

Telegram бот для обработки обратной связи от пользователей приложения Gainly.

## Команды

- `/start` - Показать информацию о боте и доступных командах
- `/feedback` - Начать заполнение формы обратной связи
- `/cancel` - Отменить заполнение формы (доступно во время заполнения)

## Форма обратной связи

Форма обратной связи состоит из 3 шагов:

1. **Выбор формата приложения**: Android или Mini App Telegram
2. **Выбор типа обратной связи**: Пожелания или Предложение новых функций
3. **Ввод текстового сообщения**

После заполнения формы сообщение отправляется разработчику
